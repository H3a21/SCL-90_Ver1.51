项目简介

本项目基于经典心理测评工具 **SCL-90（症状自评量表）** 实现，用于评估个体的心理健康状况。

量表共包含 **90 个项目**，覆盖广泛的精神症状学内容，包括：
感觉
情感
思维
意识
行为
生活习惯
人际关系
饮食与睡眠等

并通过 **10 个因子** 对不同维度的心理症状进行综合评估。
